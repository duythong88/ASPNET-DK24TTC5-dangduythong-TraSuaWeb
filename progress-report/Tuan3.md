### ✅ Nội dung đã thực hiện

Trong tuần thứ ba, em tập trung hoàn thiện hệ thống và tối ưu các chức năng đã xây dựng, cụ thể:

- Hoàn thiện chức năng quản trị (Admin):
  + Quản lý sản phẩm (thêm, sửa, xóa)
  + Quản lý danh mục sản phẩm
  + Quản lý đơn hàng
  + Cập nhật trạng thái đơn hàng (chờ xác nhận, đã xác nhận, đã giao)

- Xây dựng chức năng tìm kiếm sản phẩm theo tên
- Cải thiện chức năng hiển thị hình ảnh sản phẩm
- Hoàn thiện giao diện người dùng:
  + Điều chỉnh bố cục trang web
  + Cải thiện menu và thanh điều hướng
  + Tối ưu hiển thị trên nhiều kích thước màn hình

- Xây dựng báo cáo doanh thu:
  + Doanh thu theo ngày
  + Doanh thu theo tháng (cơ bản)

- Kiểm tra và sửa lỗi toàn hệ thống:
  + Lỗi giỏ hàng
  + Lỗi đặt hàng
  + Lỗi hiển thị dữ liệu

- Tối ưu hiệu năng hệ thống:
  + Giảm truy vấn không cần thiết
  + Sắp xếp lại code cho dễ hiểu và dễ bảo trì
  + Chụp hình từng giao diện chuẩn bị đưa vào báo cáo đồ án.
